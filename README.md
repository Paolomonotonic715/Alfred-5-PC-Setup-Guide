## 🔍 Solving installation errors

Users often face issues when setting up productivity tools on Windows 11. Check these items if the process stalls:

**Blocked installation messages**
Windows might flag the installer as unrecognized. Click "More info" on the blue prompt window and select "Run anyway" to proceed. This occurs because the software is new and lacks a wide reputation score from Microsoft.

**Permissions denied**
Verify that you run the installer as an administrator. Right-click the installation file and select "Run as administrator." 

**Conflicts with other launchers**
If you already have a tool like PowerToys Run or Flow Launcher installed, the hotkeys might conflict. Disable the hotkey in the other application settings to allow this tool to take control.

**Missing dependencies**
Ensure you have the latest .NET Runtime installed from the official Microsoft support pages. Most modern Windows utilities require these libraries to function correctly.
